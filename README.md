# Youth Market Notes

A public learning log by a 14-year-old who is fascinated by how markets work.
Every day I read one real English financial article and write my own reflection,
and I keep a diary of a **simulated** (virtual-money) portfolio.

🌐 Live site: https://zhiyenotes.com

## What's inside

- **📰 Daily Market Notes** — I read a real English financial article each day and
  write a short reflection in English: what I learned and what I think.
- **📈 Simulated Portfolio Diary** — a journal of imaginary trades using virtual money
  only, to practice thinking about risk and reasoning.

## Two rules I keep

- **Copyright** — I only link to the original article and share my own reflection.
  I never repost a full article.
- **Virtual money only** — the portfolio is a learning simulation. No real trading,
  no real money. Nothing here is investment advice.

## Tech

Built with [Vite](https://vite.dev/) + [React](https://react.dev/), and deployed on
Cloudflare Pages. I'm learning to build and understand every part of it myself.

### Run it locally

```bash
npm install      # first time only
npm run dev      # start the dev server → http://localhost:5173
npm run build    # build for production → dist/
```
